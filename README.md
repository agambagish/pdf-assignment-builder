# PDF Assignment Builder

A simple client-side web app that helps students generate a proper **assignment PDF** with a formatted **front page**.

Students can fill in their details, generate a front page from a template, upload images or a PDF, arrange the pages, and export the final document.

Everything runs **entirely in the browser** - no backend or data storage required.

## Features

- Generate assignment **front page from templates**
- Replace placeholders like `<NAME>`, `<ROLL>`, `<SUBJECT>`
- Upload **images (JPG / PNG)** as pages
- Upload an **existing PDF**
- **Drag and drop** to arrange page order
- Preview pages before export
- Add a **title to the final PDF**
- Export a **merged PDF** ready for submission

## How It Works

1. Select a **front page template**
2. Fill in your assignment details
3. Generate the **front page**
4. Upload:
    - Images (each image becomes a page), or
    - A PDF file
5. Arrange the pages using **drag and drop**
6. Enter a title for the final PDF
7. Export and download the **final merged PDF**
